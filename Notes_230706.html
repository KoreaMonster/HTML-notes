Hypertext Markup Language(HTML) is the standard markup language for creating web pages and application. Web browers receieve HTML documents from a web server or from local stroage and render them into multimedia web pages. HTML discribes the structure of a web page semantically and originally included cues for the appearance of the document.

HTMl elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
